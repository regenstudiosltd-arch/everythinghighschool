## FIGMA MAKE PROMPT – GHANA SHS SCHOOL DIRECTORY

Design a premium **School Directory** section for the **Aneres Everything High School (AES)** website that contains **every officially recognized Senior High School in Ghana**. The School Directory should help students and parents easily explore schools, compare them, and select one during the prospectus purchasing process.

**Do not hardcode the schools into the design.** Instead, design the UI to load schools dynamically from a database that the admin can update annually with the latest Ghana Education Service (GES) school list.

---

Create a modern, visually appealing page with a hero banner.

# User Experience Goal

The School Directory should become the most comprehensive and user-friendly SHS database in Ghana. Parents and students should be able to discover schools, compare options, understand what each school offers, and seamlessly continue to purchasing a prospectus or admission package. The design should feel premium, informative, and easy to navigate while supporting future updates as new schools are added or existing information changes.
