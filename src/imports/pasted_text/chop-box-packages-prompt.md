Here is a **Figma Make prompt** specifically for implementing the **Chop Box Packages** section. It instructs Figma to build a premium UI/UX rather than just displaying a list of items.

---

# FIGMA MAKE PROMPT – CHOP BOX PACKAGES

Design a premium, modern, responsive "Chop Box Packages" section for the **Aneres Everything High School (AES)** website. This section should be one of the website's main selling points and should feel similar to configuring a product bundle on Amazon or Apple.

Use a clean white background with Royal Blue as the primary color and Gold as the accent. The interface should be visually appealing, easy to understand, and optimized for both desktop and mobile.

---

## Section Header

Title:

**Choose Your Chop Box Package**

Subtitle:

*"Whether you're looking for a simple starter pack or a fully stocked hostel package, we've got you covered. Select a package or customize one to fit your budget and preferences."*

---

## Package Cards

Display three premium cards side-by-side (desktop) and as swipeable cards (mobile).

Each card should contain:

* Package Name
* Package Tagline
* Attractive package illustration or image
* Number of included items
* Price placeholder
* "Most Popular" badge (Deluxe)
* "Best Value" badge (Premium)
* "Recommended" badge (Standard for first-year students)
* Expand to View Items
* Select Package button

# Final Design Goal

The Chop Box section should feel like a premium online shopping experience rather than a simple package list. It should encourage parents and students to compare packages, understand the value of each option, and confidently customize their own Chop Box before checkout. The design should be visually rich, intuitive, and conversion-focused, seamlessly matching the overall AES brand.
